============
Installation
============


Stable release
--------------

To install jasmin_obj_store_policy_mgmt, run this command in your terminal:

.. code-block:: console

    $ pip install jasmin_obj_store_policy_mgmt

This is the preferred method to install jasmin_obj_store_policy_mgmt, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for jasmin_obj_store_policy_mgmt can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/philipkershaw/jasmin_obj_store_policy_mgmt

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/philipkershaw/jasmin_obj_store_policy_mgmt/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/philipkershaw/jasmin_obj_store_policy_mgmt
.. _tarball: https://github.com/philipkershaw/jasmin_obj_store_policy_mgmt/tarball/master
